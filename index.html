How to Use Breaker Labels

Overview

This tool generates a printable PNG label strip for the circuit breaker panel in a Cessna aircraft. Configure the strip dimensions and appearance on the left, add your breakers in the list, then export a high-resolution PNG ready for printing.

Image Size

Width / Height (in)
The physical size of the final printed image in inches. Set these to match the exact space available on your panel.
DPI
Dots per inch for the exported PNG. 300 DPI is standard for high-quality printing. Higher values produce larger files with finer detail.
Breaker width (in)
The default width of each breaker cell. Typical Cessna breakers are 0.5". This value is applied to all new breakers you add and to any existing breakers that haven't been individually customized.
Default gap (in)
The default space between each breaker cell. Adjusting this automatically updates the Default center to center value.
Default center to center (in)
The distance from the center of one breaker to the center of the next. Equal to Breaker width + Default gap. Changing this recalculates the gap automatically. Useful when you have a known center-to-center measurement from the physical panel.
End margins (in)
Empty space added before the first breaker and after the last breaker.
Top / Bottom margin (in)
Vertical padding inside the strip, pushing cell content away from the top and bottom edges.

Appearance

Background
The fill color of the overall strip. Check Transparent to export with a transparent background (useful for layering in other software).
Cell fill
The fill color inside each individual breaker cell. Check Transparent to leave cells unfilled.
Text color
The color of the label text drawn inside each cell.
Border color / Border width
The color and pixel thickness of the outline drawn around each cell. Set width to 0 to remove borders.
Font
The typeface used for all labels. Fonts shown in full opacity are installed on this computer; dimmed fonts are not available and will fall back to a system default.
Text padding (%)
How much space to leave between the text and the cell edges, as a percentage of the cell size. Higher values shrink the text.
Text rotation
The angle at which labels are drawn inside their cells. 90° (vertical) is the most common choice for narrow breaker strips, letting you read the labels top-to-bottom.

Breaker List

Each row in the list represents one circuit breaker on the strip.

Label text
The text printed inside the cell. Press Enter (or type \n) to insert a manual line break. Otherwise the app automatically finds the best one- or two-line split to maximize font size.
Gap
The space before this specific breaker, overriding the default gap. Use the − / + buttons to adjust in 0.05" steps.
Width
The cell width for this specific breaker, overriding the default breaker width. Useful when a single breaker takes more physical space on the panel.
Size
A manual font size adjustment (−9 to +9 steps) applied on top of the auto-fitted size. Handy for nudging one label larger or smaller without affecting others.
↺ Revert
Appears when a breaker's gap or width differs from the current defaults. Click it to reset that breaker back to the global defaults.
Reordering
Drag the ⠿ handle on the left of any breaker card to reorder it within the list.
Load Preset
Replaces the current list with a set of common Cessna 172 breaker labels as a starting point.
Clear All
Removes all breakers from the list after a confirmation prompt.

Toolbar

Export PNG
Downloads the strip as a full-resolution PNG at the configured DPI. The measurement overlay is never included in the export — only the clean label strip.
Save Settings
Downloads all current settings (image size, appearance, and the full breaker list) as a .json file. Use this to save your work and reload it later.
Load Settings
Loads a previously saved .json settings file, restoring the complete configuration including all breakers.
📏 Measurements
Toggles a dimension overlay above the preview canvas. It shows the total strip width, the center-to-center distance between every adjacent pair of breakers (teal), and the individual gap and cell widths with color-coded labels. Labels that are too narrow to fit inline are lifted above with a leader line.

Overflow Warning

A red banner appears when the total span of your breakers (including gaps and margins) exceeds the configured strip width. To resolve it: reduce breaker widths, reduce gaps, reduce end margins, or increase the strip width.

Tips

  • Use Save Settings frequently — the app has no auto-save and your work is lost on page refresh.
  • Enable Measurements while designing so you can verify the center-to-center spacing matches your physical panel before printing.
  • Print at 100% scale (no "fit to page") to preserve the exact physical dimensions.
  • For transparent backgrounds, use a PNG-aware application when printing or editing the exported file.
